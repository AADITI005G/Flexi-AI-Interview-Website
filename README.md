# AI Powered interview (Fullstack Starter)

## Overview
Starter fullstack project with authentication for an AI Powered interview.
Includes:
- Backend: Node.js + Express + MongoDB + JWT authentication
- Frontend: React (Vite) + Axios + Context-based auth

## Folder structure
See the repo's fullstack-auth-app/ structure (frontend, backend).

## Setup

### Backend
1. cd backend
2. cp .env.example .env and fill values (MONGO_URI, JWT_SECRET).
3. npm install
4. npm run dev (requires nodemon) or npm start

### Frontend
1. cd frontend
2. npm install
3. create .env with VITE_API_BASE=http://localhost:5000/api if needed
4. npm run dev (runs on port 5000 by default)
